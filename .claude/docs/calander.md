## 캘린더 (월간 통계용)

- 구현 방식: Tailwind Grid 직접 구현 (외부 라이브러리 미사용)
- 히트맵 색상 단계: 활동 없음(gray-100) → 1h 미만(카테고리색 200) 
  → 1~3h(300) → 3~5h(400) → 5h 이상(500)
- 셀 내부 구성:
  ┌─────────────┐
  │ 1     3h20m │  ← 날짜 + 총 시간
  │ ████▓▓░░░░░ │  ← 카테고리별 비율 바
  └─────────────┘
- 오늘: ring-2 ring-blue-500 테두리
- 선택된 날짜: bg-blue-50 강조
- 주말: 날짜 숫자 text-red-400
- 빈 날짜(이전/다음 달): opacity-30
- 클릭 시: 일별 타임 트리 뷰로 전환
